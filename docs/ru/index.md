# AgentPinBoard — Документация (русский)

AgentPinBoard — Python-библиотека, дающая LLM-агенту **граф фактов** в
качестве рабочей памяти на время одной сессии.

Агент дёргает тулы, ответы тулов автоматически разворачиваются в
ноды графа (`FactNode` / `EventNode`), а LLM читает граф через
небольшой набор готовых тулов: `explore`, `timeline`, `graph_summary`,
`search_nodes`, `what_have_i_done`.
